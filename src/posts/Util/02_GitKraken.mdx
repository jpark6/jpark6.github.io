## 🎬  소개
- Mac, Window, Linux에서 사용할 수 있는 Git GUI 툴이다.
- <a href="https://www.electronjs.org/" target="_electron">Electron</a>으로 만들어진 앱이다.
- 사용해본 다른 Git GUI 툴로는 <a href="https://desktop.github.com/" target="_githubdesktop">Github Desktop</a>과 <a href="https://www.sourcetreeapp.com/" target="_sourcetree">Source Tree</a>, <a href="https://git-fork.com/" target="_fork">Fork</a>가 있다.
- GitKraken을 사용하는 이유는 이름이 멋있고 UI가 깔끔해서이다.

<br />

## ❓ 사용 이유
- terminal에서 commit, push, pull 등의 명령어를 사용하는 것은 별로 불편함이 없다.
- 그러나 여러 파일의 변경사항을 볼 때와, commit 내역을 볼 때는 GUI 프로그램이 더 편하다.
- terminal에서 `git diff` 명령어 입력 하면 여러 파일들의 변경사항이 세로로 스크롤되서 나온다.  
- 특정 파일들만 stage하기 위해서는 `git add [디렉토리명]/[파일명]`을 일일히 입력해줘야 한다.  
<img src={gitKrakenGitDiff} alt="git diff" />  
- git GUI 툴에서는 파일별로 변경사항을 볼 수 있고, 파일별로 stage를 할 수 있다.
  <img src={gitKrakenGUIDiff} alt="gui diff" />
