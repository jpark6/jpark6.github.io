## JavaScript란?(What is Javascript?)
- 객체 기반의 스크립팅 프로그래밍 언어 웹브라우저 내에서 주로 사용.
- Node.js 같은 런타임 환경과 같이 서버 프로그래밍에서도 사용.
- Electron으로 데스크탑용 앱 개발에서도 사용 가능  
  + <a href="https://www.electronjs.org/apps" target="electron">[Electron을 사용중인 앱들]</a> 대표적인 앱 : Atom, Visual Studio Code, Slack, ...
- Javascript의 발전 배경에는 브라우저의 발전이 상당한 영향을 끼쳣다.
- 그러므로 Javascript 및 브라우저의 역사에 대해 알아보자.
<br />

## 1993년
### 모자이크 브라우저(Mosaic Browser)  
<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/e/ea/NCSA_Mosaic_Browser_Screenshot.png/600px-NCSA_Mosaic_Browser_Screenshot.png" alt="mosaic browser" />
<br />

- 최초의 대중적인 그래픽 웹 브라우저  
- 일리노이대학교의 NCSA연구소의 대학생이었던 마크 앤드리슨(Mark Andereessen)과 에릭 비나(Eric Bina)가 공동 개발함.
- 팀 버너스리가 만든 WorldWideWeb 브라우저가 텍스트 위주였던 것과 달리, 이미지를 표시할 수 있는 최초의 그래픽 웹 브라우저

## 1994년  
### 넷스케이프 네비게이터(Netscape Navigator)  
<img src="https://i.ytimg.com/vi/qKPO3Xk4DYw/hqdefault.jpg" alt="Netscape Navigator" />
<br />

- 개발 책임자였던 마크앤드리슨이 동료들을 데리고 넷스케이프 커뮤니케이션이라는 회사 설립
- 이 기업의 브라우저 내부 코드명이 모질라였음(Mosaic & Godzilla)
- 초기 인터넷 사용자들의 폭발적인 반응을 얻어 약 80%의 시장점유율 차지
- 1990년대 후반 Internet Explorer와의 싸움에서 밀려 점유율을 회복하지 못하고 개발 중단됨.

## 1995년
### 모카(Mocha), 라이브스크립트(LiveScript), 자바스크립트(JavaScript)
- 넷스케이프 커뮤니케이션은 브렌던 아이크(Brendan Eich) 영입하여 브라우저에 내장할 스크립트 언어 개발을 시작함  
- 원래 모카(Mocha)라는 이름으로 개발되었으나 공식적으로는 라이브스크립트(LiveScript)로 불렸으며 나중에 이름이 자바스크립트로 변경됨.  
- 12월 넷스케이프 베타 2.0 베타 3에서 자바스크립트라는 이름을 채용함  
- 자바스크립트가 들어간 넷스케이프 공식버전은 1996년 3월에 출시함.

## 1996년
### ECMAScript
- 11월 넷스케이프에서 표준화를 위해 Javascript 기술 규격을 ECMA라는 비영리 표준 기구에 제출
- ECMAScript 1은 1997년 6월에 출판되었고 2021년 6월 15일 현재 ECMAScript 2021(ES12) 까지 나와 있음.

## 1999년
### XMLHTTPRequest(XHR)
- 3월 Internet Explorer 5.0에 처음으로 탑재됨.
- 비동기적으로 작업을 수행함으로써 페이지 전체가 아닌 일부분만 변경이 가능해짐.

## 2004년  
### AJAX(Asynchronous JavaScript and XML)  
- 많은 사이트를 찾아봤으나 XHR과 AJAX의 차이점이 크게 뭔지 잘 모르겠음.
- JQuery에서 ajax로 사용하고 VanillaJS 사용시 XHR을 사용하는 정도로만 알고있다. 
- 4월1일 구글에서 발표한 gmail과 2005년 2월 8일에 발표한 구글지도가 아무런 플러그인 없이 AJAX로 구현된 것으로 밝혀져 화제가 되었음.
